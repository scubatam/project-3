Box Office Predictions Made Using Machine Learning

The intent of the project was inspirated by a Machine Learning competition on Kaggle. This competition has the goal to predict the worldwide revenue for 4,398 movies using Machine Learning. At last count, this competiton has 398 teams competing, made up of 437 competitors who have already submitted a total of 4,165 entries. This competition is scheduled to end on May 30, 2019.

  • The metadata was obtained from The Movie Database (TMDB) and provided by Kaggle as part of the competition. The metadata contained movie ID, and the data points which included cast, crew, plot keywords, budget, posters, release dates, languages, production companies, and countries in which the movies were produced all in a JSON format within a CSV file. The team opted not to use all data points in the predictive analysis.

  • The movie details,credits and keywords have been collected from the TMDB Open API.

  • TMDB is used by more than 200,000 developers. On an average day, TMDB will have over a million users and process over 3 billion requests.

  • Every year since 2008, the number of contributions to our database has increased. With over 200,000 developers and companies using our platform, TMDB has become a premiere source for metadata.

  • TMDB is international and support 39 languages. On an average day, they are used in over 180 countries.
